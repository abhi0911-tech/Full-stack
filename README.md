# Entertainment App

A modern web application for discovering, browsing, and bookmarking movies and TV series. 

## Features

- **Browse Movies & TV Series**: Explore a vast collection of movies and TV series with beautiful poster displays
- **Search**: Find your favorite content with a powerful search functionality
- **Detail Pages**: View comprehensive information including plot, cast, genres, ratings, and similar recommendations
- **Bookmarks**: Save your favorite movies and TV series locally in your browser
- **Similar Recommendations**: Discover similar content based on what you're viewing
- **Responsive Design**: Fully responsive UI that works on desktop, tablet, and mobile devices
- **Dark Theme**: Eye-friendly dark theme throughout the application

## Tech Stack

- **Frontend**: React 18 with Vite
- **Styling**: Tailwind CSS + PostCSS
- **Routing**: React Router v6
- **API**: TMDB API v3
- **Storage**: Browser LocalStorage for bookmarks

## How to Use

1. **Browse Content**: Navigate to Movies or TV Series pages using the sidebar
2. **Search**: Use the search bar to find specific content
3. **View Details**: Click on any card to view detailed information
4. **Bookmark**: Click the bookmark button to save to your favorites
5. **Check Bookmarks**: Visit the Bookmarks page to view all saved items
